Welcome to Toktogi Plus !

STEP 1

Click the Toktogi action button and click “Enable Toktogi”

STEP 2

Hover your mouse over the text below to test it out.

똑똑이는 한국어 공부 사전 입니다.

STEP 3

Play around with configuration page at Option page (Or simply manually navigate to extension page and click Option)

You can save vocab list by pressing on the Icon ,Either as 'tsv' format , or 'direct import to Anki App'(Anki-Connect add-on require)

Press '[' to turn on aggressive vocab recognition mode (match more words, but high false positive & cpu intensive)

Press ']' to turn off aggressive vocab recognition mode

Press alt + m to open Option Page

Experiment and get familair with buttons to see what it can do

There are buttons to lookup words on naver,google translate, papago translate (on highlight texted)

There are buttons to perform wildcard search word for highlighted text. e.g.

  • For example, try highlighting the word 경로
  • clicking on "a*" button will match all entries that starts with "경로". i.e. 경로하다, 경로회 , 경로잔치, etc
  • clicking on "*a*" button will match all entries that contains with "경로"
  • clicking on "a*다" button will match all entries that contains with "경로", and must end with "다" i.e. 경로하다
  • You can also reverse search. that is, if you highlight non korean word such as "office", then clicking on "a*" button will match all entries that has english definition containing "office"

There is button to save vocab in TSV format, or Import directly to Anki (Anki-connect plugin desktop client required)

You might need to restart browser for configuration changes to take effect

If Toktogi Plus popup doesn't work. try refreshing with F5
