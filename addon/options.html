0.6.7c2 - Bonus Features

  • Toggle for improving Conjugated Word Recognition, but cost more CPU power. Default is On
  • New Feature: AnkiConnect Add-on Integration
    • In Option Menu, Choose Whether to Save Vocab List to 'TSV' File, or export directly to Anki /li>
    • Requirement for direct Anki Export:
      1. AnkiConnect Add-on must have been installed on Anki
      2. Anki Application must be running.
      3. Easy way to check if you did everything correctly is to go to your browser and visit http://127.0.0.1:8765/ . Make sure firewall isn't blocking this
      4. Anki Import Format is hard coded:
      5. deckname : "Vocab_Toktogi"
      6. model/Note Name : "Toktogi"
      7. Field Names, Make sure to have the following(In Any Order, As Long as "Korean_Vocab" is the first field):
        1. "Korean_Vocab"
        2. "Meaning_E"
        3. "Context_Sentence"
        4. "Source_Title"
  • Choose populated Dict entry to save using KeyDown 1,2,3,4
  • Easter Egg: Expect Toktogi to not work very well in private browsing mode i.e. Changes Made to Option.ui Needs firefox-restart to take effect.
