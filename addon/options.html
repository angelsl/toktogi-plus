Change log

0.6.7b1 - Bonus Features

  • Adds clunky Option Menu with button to show currently saved Vocab List
  • Press 's' | Or Click on the + button Next to Vocab To Save Vocab to List in temporary Memory (Vocab List will be reset after closing firefox)
    • Vocab List Template is As Follow:
    • 1.{Vocab}, 2.{Meaning + Hanja}, 3.{Context Sentence}, 4.{Source Html Title}
    • TODO: Hanja will have its own field. & Ability to re-order Fields during Export.
    • TODO: Option To choose between saving {Context Sentence} (Shorter) {Context CurrentNode} (Longer, like Paragragh)
  • Press 'x' To Export Saved Vocab List to TSV File (For importing to Anki)
  • Press 'u' To Upload/Store Saved Vocab List in Persistent Local Storage (Vocab List Will not be reset after closing firefox)
  • Press 'r' To Retrieve Saved Vocab List from Persistent Local Storage
  • Press 'p' To Purge/delete Saved Vocab List from Persistent Local Storage
  • Press 'm' To Open Option menu Alternatively, You can manually access Toktogi's menu from Firefox Addon >Extension Tab
  • Re-Adjust Dictionary Pop-up Box so it doesn't appear too far out from the highlighted word
  • Quick fix to make Toktogi-Plus Add-on (Somewhat) Compatible with Android.
